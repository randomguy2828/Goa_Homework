inline elements are elements that do not start on a new line and only occupy as much width as necessary, block level elements are elements that start on a new line and take up the full width available,

box model is a fundametal thing in web designing which describes how elements are structured

content is the centre where it displays actual content, padding is an area that surrondors content and provides space between content and border, border encloses both content and border and can be styled, margin creates space between different elements.
